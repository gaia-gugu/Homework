# Homework Stars ⭐

A cute homework and task tracker for kids aged 7–11. Built with React + Vite + TypeScript.

## Features

- 🐾 **Setup screen** — enter your name and pick from 30 animal avatars
- 🈶 **Subjects** — Chinese, English, Math, Science, Art, Music, PE, and more
- 📋 **Filter tabs** — Today / All / Done with task counts
- ⭐ **Star rewards** — earn +1 or +2 stars per completed task
- 🎉 **Celebration animation** — animated star burst on task completion
- 🔴 **Overdue highlighting** — tasks turn red when past due
- 💾 **Persistent** — all data saved to localStorage
- 📱 **iPad-friendly** — 48px+ touch targets, safe-area support, responsive layout

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
